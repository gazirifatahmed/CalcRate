## 🚀 Overview

**CalcRate** is a lightweight, offline-capable web application that combines a real-time currency converter with a fully functional calculator. Built on **MVVM architecture**, it ensures clean code separation, easy maintenance, and scalability.

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 💵 **Live Conversion** | USD → BDT / INR / EUR |
| 🧮 **Smart Calculator** | Basic ops (+, -, ×, ÷, %) |
| 🌙 **Night Mode** | Eye-friendly dark theme |
| 🔊 **Sound Feedback** | Optional haptic/audio |
| 📜 **Rate Agreement** | Transparent exchange terms |
| 🔌 **Offline Ready** | No backend required |
